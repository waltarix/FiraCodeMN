## FiraCodeMN

### Prerequisites

- [GNU Make](https://www.gnu.org/software/make/)
- [Docker](https://www.docker.com/)

### Usage

```sh
$ make by-docker

$ tree -P 'FiraCodeMN*' ./build
./build
├── FiraCodeMN-Bold-Italic.ttf
├── FiraCodeMN-Bold.ttf
├── FiraCodeMN-Italic.ttf
├── FiraCodeMN-Regular.ttf
├── FiraCodeMNStyled-Bold-Italic.ttf
├── FiraCodeMNStyled-Bold.ttf
├── FiraCodeMNStyled-Italic.ttf
└── FiraCodeMNStyled-Regular.ttf

0 directories, 8 files
```
